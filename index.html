功能要点

从编排到执行、从配置到复盘，PinableAgents 把 AI 开发工作流整理成可落地的标准动作。

统一编排：Claude Code + codeagent-wrapper

编排者负责计划与上下文收集，执行层负责代码修改、测试执行与会话恢复。

工作流家族

不同场景调用不同工作流，形成可重复的协作节奏。

do（推荐）

7 阶段功能开发，强制澄清 + 评审闭环，适合常规功能交付。

omo

风险信号驱动路由，适合未知定位的 bug 或跨模块重构。

bmad

企业级敏捷协作，包含产品、架构、开发、测试多角色。

sparv / requirements

轻量化流程，适合快速原型与需求到代码的快速闭环。

模块与场景

核心架构

编排与执行分层，确保每一步都可追溯、可复用。

编排层：Claude Code

负责规划、上下文收集、验证，输出结构化任务给执行层。

执行层：codeagent-wrapper

统一 CLI，连接 Codex/Claude/Gemini，支持会话恢复与并行。
