## 🌟 项目简介

Pinable Game Prototypes 是一个现代化的游戏原型展示平台，用于展示和体验各种创意 HTML5 游戏原型。

**在线访问：** [https://pinable.cc](https://pinable.cc)

### ✨ 特性

- 🎨 **现代化设计** - 渐变色、玻璃态效果、流畅动画
- 🔍 **实时搜索** - 快速找到您想玩的游戏
- 🏷️ **标签筛选** - 按类型、难度筛选游戏
- 📱 **响应式设计** - 完美适配桌面、平板、手机
- ⚡ **高性能** - 优化的加载和渲染
- 🎯 **即时体验** - 点击即玩，无需下载
